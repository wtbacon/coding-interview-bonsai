# Smallest Subarray with a Greater Sum

- Pattern: Sliding Window

## Problem

Given a positive integer array `arr` and a positive integer `S`, find the length of the smallest contiguous subarray whose sum is greater than or equal to `S`. Return 0 if no such subarray exists.

## Notes

- Time: $O(N)$
- Space: $O(1)$
- Approach: Sliding Window
